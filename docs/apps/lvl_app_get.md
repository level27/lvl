# lvl app get

Shows a list of all available apps.

```
lvl app get [flags]
```

### Examples

```
lvl app get -f FilterByName
```

### Options

```
  -f, --filter string   How to filter API results?
  -h, --help            help for get
  -n, --number int      How many things should we retrieve from the API?
```
