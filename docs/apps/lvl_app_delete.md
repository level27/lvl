# lvl app delete

Delete an app

```
lvl app delete [flags]
```

### Examples

```
lvl app delete NameOfMyApp
```

### Options

```
  -h, --help   help for delete
  -y, --yes    Set this flag to skip confirmation when deleting an app
```
